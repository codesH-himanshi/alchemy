# Alchemy Unit Converter
A sleek unit conversion tool that transforms measurements like magic  

## Features
- 🌡️ Temperature (Celsius ↔ Fahrenheit)
- ⚖️ Weight (Pounds ↔ Kilograms)
- 📏 Distance (Miles ↔ Kilometers)
- Dark mode interface
- Responsive design

## Tech Stack
**Frontend**:  
HTML5 | CSS3 | JavaScript  

**Backend**:  
Node.js | Express
